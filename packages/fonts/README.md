# Fonts

This package contains the fonts for Autodesk HIG.

## Getting started

### Install the package

```bash
yarn add @hig/fonts
```

## Basic usage

### Import stylesheets

```js
/* ArtifaktElement font w/ weights used in the design system (400, 600, and 800)  */
import "@hig/fonts/build/ArtifaktElement.css";
```

```scss
/* ArtifaktElement font w/ weights used in the design system (400, 600, and 800)  */
@import "~@hig/fonts/build/ArtifaktElement.css";
```
